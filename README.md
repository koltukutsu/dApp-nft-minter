# nft-minter-dapp

Prerequisites

- Deployed TBTKNFT contract address see [nft-contract](https://bag.org.tr/proje/ether/nft/nft-contract)

Install dependencies

```sh
$ npm install
```

Build 

```sh
$ npm run build
```

Run development server
``` sh
$ npm start
```
